## Use

`kure config edit`

## Description

Edit the current configuration file.

#### Text editors commands
*Editor*: *command*
```
Vim: vim
Neovim: nvim
Emacs: emacs
Nano: nano
Visual Studio Code: code
Sublime Text: subl
Atom: atom
Coda: coda
Notepad: notepad
Notepad++: notepad++
...
```

## Flags 

No flags.

### Examples

Edit the configuration file being used:
```
kure config edit
```
