## Description

Card operations.

## Flags

No flags.

### Subcommands

- kure card **add**: Add a card to the database.

- kure card **copy**: Copy card number or cvc.

- kure card **list**: List a card or all the cards from the database.

- kure card **rm**: Remove a card from the database.
