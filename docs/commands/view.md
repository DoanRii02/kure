## Use

`view [-p port]`

## Description

Display all entries on a local http server.

## Flags 

|  Name     | Shorthand |     Type      |    Default    |       Usage           |
|-----------|-----------|---------------|---------------|-----------------------|
| port      | p         | uint16        | 4000          | Set server port       |

### Examples

View entries:
```
kure view -p 8080
```
