## Description

Wallet operations.

## Flags 

No flags.

### Subcommands

- kure wallet **add**: Add a wallet to the database.

- kure wallet **copy**: Copy wallet public key.

- kure wallet **list**: List a wallet or all the wallets from the database.

- kure wallet **rm**: Remove a wallet from the database.
